ACCESS THE ADMIN PANEL:
  1. Main Dashboard:  http://yoursite.com/admin
  2. Netflix Config:  http://yoursite.com/admin/netflix

WHAT YOU CAN DO:

  ✓ Edit Netflix Subscription Plan Names
  ✓ Change Plan Prices (Rs. 350, Rs. 499, Rs. 999, etc.)
  ✓ Update Plan Durations (1 Month, 3 Months, 6 Months, etc.)
  ✓ Add/Remove/Edit Features for Each Plan
  ✓ Change Plan Descriptions (with Netflix subscription keywords)
  ✓ Update Buy Links (which products page to link to)
  ✓ Mark Plans as "Best Value"
  ✓ Edit Page Title and Description
  ✓ Manage Action Buttons (text, link, color)
  ✓ View All SEO Keywords

HOW IT WORKS:

  1. Make Changes → Click "Save All Changes" → Done!
  2. Changes save to browser (localStorage)
  3. Changes appear instantly on Netflix in Nepal page
  4. No database or code changes needed
  5. No website downtime

EXAMPLE EDITS:

  Change Netflix Price:
    - Go to Admin Panel → Netflix Config
    - Find "Standard" plan
    - Change price from "Rs. 499" to "Rs. 599"
    - Click "Save All Changes"
    - Visit /netflix-in-nepal to see new price

  Add a Feature:
    - Find Netflix Plan in admin panel
    - Click "Add Feature" button
    - Type: "Netflix subscription with 4K Ultra HD"
    - Click "Save All Changes"
    - Feature appears on website instantly

  Update CTA Button:
    - Scroll to "Action Buttons" section
    - Change button text
    - Update link URL
    - Click "Save All Changes"
    - Button updates on all pages using it

IMPORTANT NOTES:

  ✓ Clearing browser data will reset to defaults

  ✓ Changes are instant (no page reload needed)
  ✓ Can preview changes in admin panel

RESET TO DEFAULTS:

  If you want to start fresh:
    1. Open browser console (F12 or right-click → Inspect)
    2. Type: localStorage.removeItem('netflix-config')
    3. Press Enter
    4. Refresh the page

TROUBLESHOOTING:

  Changes not showing?
    → Refresh the Netflix page in a new tab
    → Check that you clicked "Save All Changes"
    → Make sure localStorage is not disabled

  Disappeared after closing browser?
    → Browser was cleared (history/cache)
    → Your changes are still in the config until you delete
