WHAT IS THIS?
  A system to manage Netflix subscription content without coding.
  Edit prices, features, buttons, and content - changes appear instantly.

HOW TO ACCESS?
  /admin/netflix ← Go here to edit Netflix content

HOW TO USE?
  1. Visit /admin/netflix
  2. Edit content (prices, features, text)
  3. Click "Save All Changes"
  4. Changes appear instantly on /netflix-in-nepal
  5. Done!

WHAT CAN YOU EDIT?

  NETFLIX PLANS (4 plans)
  ├─ Name: Mobile, Basic, Standard, Premium
  ├─ Price: Rs. 350, Rs. 499, Rs. 999, etc.
  ├─ Duration: 1 Month, 3 Months, etc.
  ├─ Features: Add/remove/edit as many as you want
  ├─ Description: Plan description text
  ├─ Buy Link: Which product page
  └─ Best Value: Mark which plan is best

  ACTION BUTTONS (up to 2 buttons)
  ├─ Text: "Buy Netflix Now" etc.
  ├─ Link: /category?search=netflix etc.
  └─ Style: Primary (amber) or Secondary (outlined)

  PAGE CONTENT
  ├─ Title: Netflix page title
  └─ Description: Netflix page description

  SEO KEYWORDS
  └─ View: 30+ keywords (read-only)

COMMON EDITS

  Change Price?
    1. Find plan in admin panel
    2. Change price: Rs. 499 → Rs. 599
    3. Click "Save All Changes"
    ✓ Price updated on website

  Add Feature?
    1. Find plan in admin panel
    2. Click "Add Feature"
    3. Type feature text
    4. Click "Save All Changes"
    ✓ Feature appears on website

  Remove Feature?
    1. Find feature in admin panel
    2. Click delete/trash button
    3. Click "Save All Changes"
    ✓ Feature removed from website

  Update Button?
    1. Go to "Action Buttons" section
    2. Edit text or link
    3. Click "Save All Changes"
    ✓ Button updated on website

  Mark Different Plan as Best?
    1. Uncheck "Best Value" for current plan
    2. Check "Best Value" for new plan
    3. Click "Save All Changes"
    ✓ Badge moves to new plan

TROUBLESHOOTING

  Changes not showing?
    → Refresh /netflix-in-nepal page
    → Check you clicked "Save All Changes"
    → Check browser console (F12) for errors

  Data disappeared?
    → Browser history was cleared
    → Your data was stored in localStorage
    → Read ADMIN_GUIDE.md for reset instructions

  Can't access admin panel?
    → Check URL: /admin/netflix
    → Enable JavaScript
    → Try different browser
    → Clear cache (Ctrl+Shift+Del)
