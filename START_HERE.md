### Task 2: Add a Feature to Netflix Plan
1. Go to `/admin/netflix`
2. Find the plan
3. Click "Add Feature" button
4. Type the feature (e.g., "Netflix subscription with spatial audio")
5. Click "Save All Changes"
6. Done! Feature appears on website

### Task 3: Remove a Feature
1. Go to `/admin/netflix`
2. Find the plan and feature
3. Click trash/delete button next to feature
4. Click "Save All Changes"
5. Done! Feature is removed from website

### Task 4: Update CTA Button
1. Go to `/admin/netflix`
2. Scroll to "Action Buttons"
3. Edit button text or link
4. Click "Save All Changes"
5. Done! Button updates on website

1. Go to `/admin/netflix`

4. Click "Save All Changes"
5. Done! Badge moves to new plan

## Troubleshooting

### Changes Don't Appear?
1. Refresh the Netflix page (`/netflix-in-nepal`)
2. Check that you clicked "Save All Changes"
3. Check browser console for errors (F12)

### Data Disappeared?
- Your browser history was cleared
- localStorage data is cleared with browser history
- Check `ADMIN_GUIDE.md` for reset instructions

### Can't Access Admin Panel?
1. Check URL: `/admin/netflix`
2. Make sure JavaScript is enabled
3. Try a different browser
4. Clear browser cache

### Buttons Don't Look Right?
1. Check button variant (Primary = amber, Secondary = outlined)
2. Check that links start with `/` or `http://`
3. Save changes again

### Need More Help?
- See: `ADMIN_GUIDE.md` - Troubleshooting section
